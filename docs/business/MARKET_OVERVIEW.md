# 云边协同空间智能平台市场方案

## 1. 产品定位

平台定位为云边协同空间智能平台，面向园区、水利、施工、交通、环保、能源、应急和无人系统等场景，提供视频接入、设备管理、边缘推理结果接入、云端训练、大模型分析、空间态势、告警处置和报表能力。

平台以空间位置为业务入口，将摄像头、边缘一体机、传感器、无人车、无人机、机器人、告警事件和 AI 分析结果统一展示到地图、区域和点位中，形成可运行、可管理、可汇报的空间智能系统。

## 2. 目标客户

- 政企园区：园区安防、巡检、人员车辆管理、风险预警。
- 水利水务：河道巡检、漂浮物识别、水体异常、闸坝设施监控。
- 施工单位：安全帽、反光衣、危险区域、施工进度、现场违规。
- 交通管理：车流、人流、违停、拥堵、逆行、道路事件。
- 环保监管：烟火、扬尘、黑臭水体、排口、裸土覆盖。
- 能源电力：通道巡检、设施异常、鸟巢、锈蚀、表计识别。
- 无人系统集成商：无人车、无人机、机器人和边缘 AI 设备统一接入。

## 3. 核心价值

- 空间化管理：设备、视频、告警和任务在地图上统一管理。
- 云边协同：边缘侧低延迟推理，云端进行训练、复核、报表和大模型分析。
- 业务闭环：从视频接入、算法任务、告警处置到样本回流和模型迭代形成闭环。
- 快速落地：采用 Docker 容器化部署，支持开发版 Windows + WSL + Docker 和生产版 Linux + Docker。
- 平台可扩展：当前支持二维地图，后续扩展三维地图和数字孪生大屏。

## 4. 差异化能力

- 流媒体核心采用 ZLMediaKit，避免自研视频底层带来的稳定性风险。
- 边缘推理采用独立 C++ 框架 `edge_infer`，云端平台负责结果接入、展示和治理。
- 云端支持 Python 训练、零样本推理、大模型问答和报表生成。
- 平台支持 DeepSeek、OpenAI、通义千问、本地大模型等供应商配置。
- 训练追踪默认采用平台内置方案，MLflow 作为可选增强，避免训练服务受 MLflow 故障影响。

## 5. 商业化形态

- 私有化项目交付：面向客户现场部署。
- 行业解决方案交付：按园区、水利、施工、交通等场景打包。
- 平台授权：按项目、设备数量、视频通道数量、边缘节点数量授权。
- 算法能力授权：按场景能力、模型包、评测和持续迭代服务收费。
- 运维服务：部署、巡检、模型更新、报表和现场支持。

## 6. 首批推荐落地

首批项目聚焦三个高价值场景：

- 安全生产：人员、车辆、安全装备、危险区域、烟火等能力。
- 交通监管：车流、人流、违停、拥堵、逆行等能力。
- 水利巡检：河道、漂浮物、水体异常、设施状态等能力。

每个场景先落地 3-5 个高频能力，形成稳定样板，再扩展到 100+ 场景能力。
