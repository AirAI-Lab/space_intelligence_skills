# 场景解决方案

## 1. 总体方案

平台以“设备接入 + 视频通道 + 算法任务 + 告警处置 + 空间态势 + 报表”为标准方案骨架。不同场景通过能力配置、模型版本、检测区域、告警规则和报表模板进行差异化交付。

## 2. 安全生产

目标：提升园区、工厂、仓储和重点区域的安全监管效率。

核心能力：

- 安全帽检测。
- 反光衣检测。
- 人员闯入。
- 人员聚集。
- 跌倒识别。
- 烟火识别。
- 消防通道占用。
- 灭火器和消防栓状态。
- 危险区域布防。

交付页面：

- 空间态势。
- 视频接入。
- 智能分析。
- 告警处置。
- 日报/月报。

## 3. 水利巡检

目标：提升河道、水库、闸坝、泵站和排口巡检效率。

核心能力：

- 河道漂浮物。
- 非法垂钓。
- 非法游泳。
- 水体异常。
- 排口异常。
- 水位异常。
- 闸门设施异常。
- 堤岸破损。

交付页面：

- 河道空间态势。
- 视频点位管理。
- 告警热力图。
- 巡检报表。

## 4. 施工安全

目标：降低施工现场违规和安全事故风险。

核心能力：

- 安全帽。
- 反光衣。
- 高空作业。
- 临边洞口。
- 危险区域闯入。
- 施工车辆。
- 物料堆放。
- 施工进度辅助识别。

交付页面：

- 工地态势图。
- 视频看板。
- 告警处置。
- 月度安全报告。

## 5. 交通监管

目标：实现道路、园区、厂区和重点通道的交通状态感知。

核心能力：

- 车流统计。
- 人流统计。
- 车辆类型识别。
- 违停。
- 逆行。
- 压线。
- 拥堵。
- 车牌识别。

交付页面：

- 道路态势图。
- 通道视频预览。
- 交通事件列表。
- 统计报表。

## 6. 环保监管

目标：提升环境异常发现、取证和处置效率。

核心能力：

- 烟火。
- 扬尘。
- 裸土未覆盖。
- 黑臭水体。
- 排口异常。
- 垃圾堆放。
- 渣土车识别。

交付页面：

- 环保态势图。
- 告警热力图。
- 事件证据。
- 整改统计。

## 7. 无人系统接入

目标：接入无人车、无人机、机器人和机器狗，形成移动感知能力。

核心能力：

- 实时位置。
- 轨迹回放。
- 视频流接入。
- 边缘推理结果上报。
- 巡检任务状态。
- 异常事件定位。

交付页面：

- 移动设备态势。
- 轨迹地图。
- 实时视频。
- 巡检事件。
