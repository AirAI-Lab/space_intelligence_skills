# 竞争优势与投标指标

## 1. 平台优势

- 云边协同架构：边缘侧实时推理，云端训练、复核、报表和大模型分析。
- 空间智能底座：设备、视频、告警、任务和报表全部关联地图点位和区域。
- 成熟流媒体底座：ZLMediaKit 负责视频接入、转发、截图、录像和播放。
- 独立边缘推理框架：`edge_infer` 采用 C++ 实现，适合边缘设备稳定运行。
- 轻量平台实现：Spring Boot 后端、Vue3 前端、Python 云端算法服务，避免重型平台过度复杂。
- 容器化部署：支持开发版和生产版部署。
- 大模型可插拔：支持网络 API、本地模型和混合模式。

## 2. 可量化指标

平台指标按项目硬件和网络条件验收：

- 视频通道接入：支持 RTSP、RTMP、HLS、HTTP-FLV、WebRTC 扩展。
- 流媒体状态：支持流上线、下线、播放、推流、截图、录像事件。
- 设备接入：支持边缘一体机、摄像头、传感器、无人系统。
- 告警闭环：支持告警证据、处置记录、误报反馈、样本回流。
- 训练闭环：支持数据集、训练任务、指标、模型版本、评测、部署。
- 地图能力：支持二维地图、设备点位、视频点位、告警定位、热力图。
- 报表能力：支持日报、月报、训练总结、告警统计。

## 3. 控标亮点

- 支持云边协同 AI 推理结果统一接入。
- 支持地图点位和空间态势展示。
- 支持视频流管理、截图、录像和播放鉴权。
- 支持大模型问答、报表生成和训练进度查询。
- 支持模型包标准化交付和边缘部署状态追踪。
- 支持 MLflow 可选接入，训练主流程不依赖 MLflow。
- 支持开发版 Windows + WSL + Docker 和生产版 Linux + Docker。

## 4. 风险控制

- 流媒体核心不自研，降低视频稳定性风险。
- 边缘推理不由云端平台重写，降低边缘框架迁移风险。
- 大模型默认只访问脱敏后的平台数据和受控工具。
- 首批场景聚焦少量高价值能力，避免一次性铺开导致交付失控。
- 旧项目文档归档管理，正式文档保持少量、准确、可维护。
