### 1.3 热阻分析

```
热阻网络：

环境温度 (Ta = 25°C)
    │
    └─> 热源 (Tj)
         │
         ├─> 结-壳热阻 (Rth_jc) = 0.5°C/W
         │    (芯片内部)
         │
         ├─> 壳-散热片热阻 (Rth_cs) = 0.3°C/W
         │    (导热硅脂)
         │
         └─> 散热片-环境热阻 (Rth_sa)
              │
              ├─> 被动散热 = 3.0°C/W
              │
              └─> 主动散热（风扇）= 1.5°C/W

总热阻：
  被动散热：Rth_total = 0.5 + 0.3 + 3.0 = 3.8°C/W
  主动散热：Rth_total = 0.5 + 0.3 + 1.5 = 2.3°C/W

温度估算：
  被动散热：Tj = Ta + P × Rth = 25 + 25×3.8 = 120°C ❌ 超标
  主动散热：Tj = Ta + P × Rth = 25 + 25×2.3 = 82.5°C ✅ 达标
```

```
- 鳍片高度：18mm

热阻计算：
  Rth = 1 / (h × A × η)
  
  h = 50 W/(m²·K)  (强制风冷)
  A = 0.028 m²     (有效面积)

  Rth = 1 / (50 × 0.028 × 0.85) = 0.84°C/W

```

```
        环境空气
           ↓

    └──────┬───────┘

    ┌──────────────┐

风量计算：
  单个风扇：Q = 0.23 m³/min
  双风扇总风量：Q_total = 2 × 0.23 = 0.46 m³/min
  
  换热能力：
  P = Q × ρ × Cp × ΔT
  P = 0.46/60 × 1.2 × 1005 × 20 = 185W
  
  结论：双风扇足以散热33.5W
```

## 六、温控算法

### 6.1 PWM控制策略

```
温控策略（滞后控制）：

温度(°C)  风扇转速  PWM占空比  噪音(dBA)
─────────────────────────────────────
< 40       0%        0%         0
40-45      0%→30%    30%        25
45-50      30%→50%   50%        30
50-60      50%→70%   70%        35
60-70      70%→90%   90%        40
> 70       90%→100%  100%       45

滞后区间：2°C（防止频繁切换）

示例：
  - 温度上升到42°C → 风扇启动30%
  - 温度下降到38°C → 风扇停止
  - 温度上升到52°C → 风扇加速到50%
  - 温度下降到48°C → 风扇减速到30%
```

### 6.2 控制算法（伪代码）

```python
class FanController:
    def __init__(self):
        self.temp_thresholds = [40, 45, 50, 60, 70]
        self.pwm_levels = [0, 30, 50, 70, 90, 100]
        self.hysteresis = 2  # 滞后2°C
        
    def get_pwm(self, current_temp, previous_temp):
        # 根据温度趋势选择阈值
        if current_temp > previous_temp:  # 温度上升
            for i, threshold in enumerate(self.temp_thresholds):
                if current_temp < threshold:
                    return self.pwm_levels[i]
            return self.pwm_levels[-1]  # 最高转速
        else:  # 温度下降
            for i, threshold in enumerate(self.temp_thresholds):
                if current_temp < threshold - self.hysteresis:
                    return self.pwm_levels[i]
            return self.pwm_levels[-2]  # 降一档
    
    def update(self, temp):
        pwm = self.get_pwm(temp, self.prev_temp)
        self.set_fan_pwm(pwm)
        self.prev_temp = temp
        return pwm
```

