# 妙算 3 5G+Mesh+北斗+AI 推理平台集成方案 V2.0

> **版本**: V2.0  
> **更新日期**: 2026-03-06  
> **优化目标**: 解决V1.0问题，优化功耗散热，避免授权频段，降低成本10-20%

---

## 一、V1.0问题回顾

### 已识别的问题

| 问题 | V1.0现状 | 影响 |
|------|---------|------|
| **主控芯片不明确** | Orin NX/Xavier NX/Orin Nano混用 | 导致功耗预算不准确 |
| **功耗目标不现实** | 目标25W，实际需35-40W | 供电和散热设计不足 |
| **缺少主动散热** | 依赖被动散热 | 高负载下过热风险 |
| **1.4GHz需授权** | 使用1.4GHz频段 | 合规风险，成本高 |
| **成本过高** | 约10,472元 | 超出预算10-20% |

### 3.2 主控芯片明确

**确认型号**: **Jetson Orin NX 16GB**

**理由**:
- 大疆妙算3官方配置为Orin NX 16GB版本
- 提供最佳的算力/功耗平衡
- 16GB内存满足AI推理需求
- 供应链稳定，价格明确

**关键参数**:
- AI算力: 100 TOPS (INT8)
- GPU: 1024 CUDA cores, 32 Tensor Cores
- TDP: 15-30W（可调）
- 典型功耗: 20-25W（中等负载）

### 3.3 频段优化方案

#### 避免使用1.4GHz的原因

| 问题 | 说明 |
|------|------|
| 需要授权 | 1.4GHz（1427-1518MHz）为专用频段，需向工信部申请 |
| 申请周期长 | 3-6个月审批时间 |
| 费用高昂 | 频谱使用费+设备认证费约2-3万元 |
| 合规风险 | 未授权使用面临处罚 |

#### 优化的频段选择

**5G通信模组频段**:
- **Sub-6GHz**: N78（3.5GHz），免授权频段
- 覆盖范围: 1-5km（视距）
- 带宽: 100MHz
- 上行/下行速率: 500Mbps/3Gbps

**Mesh通信模组频段**（三频段备选方案）:

| 频段 | 频率范围 | 优势 | 劣势 | 推荐场景 |
|------|---------|------|------|---------| 
| **2.4GHz** | 2400-2483.5MHz | 穿透能力强，成本低 | 干扰大，带宽低 | 城市密集区 |
| **5.8GHz** | 5725-5850MHz | 带宽高，干扰小 | 穿透弱，距离短 | 开阔地带 |
| **433MHz** | 433.05-434.79MHz | 距离远，穿透强 | 带宽低 | 长距离组网 |

**推荐配置**:
- **主频段**: 5.8GHz（主要Mesh组网）
- **备用频段**: 2.4GHz（室内/城市环境）
- **双频段切换**: 根据环境自动选择最优频段

**北斗定位频段**:
- B1I: 1561.098MHz（民用）
- B2b: 1207.140MHz（精密单点定位）
- 免授权，直接使用

- UART 1: NMEA数据输出

## 六、PCB设计要点

### 6.1 电源分区设计

**分区原则**: 妙算3电源区、5G模块电源区、Mesh/北斗电源区严格分区

```
电源分区布局：

┌─────────────────────────────────────┐
│  电源输入（5V/3A）                  │
└──────────────┬──────────────────────┘
               │
        ┌──────┴──────┬──────────────┐
        ▼              ▼              ▼
┌───────────────┐ ┌──────────────┐ ┌──────────────┐
│  妙算3电源区  │ │  5G电源区    │ │  低功耗区    │
│  5V/3A       │ │  5V/2.5A     │ │  3.3V/1A    │
│  独立GND     │ │  独立GND     │ │  Mesh/北斗   │
└───────────────┘ └──────────────┘ └──────────────┘
```

**设计要点**:
- 5V主电源走线宽度 ≥ 2mm
- 3.3V电源走线宽度 ≥ 1mm
- 单点共地，接地电阻 ≤ 0.1Ω
- 每个电源区独立去耦电容

### 6.2 信号完整性设计

**USB 3.0信号**:
- 差分对严格等长（误差 < 5mil）
- 阻抗控制：90Ω±10%
- 走线间距：3W原则
- 远离射频天线 ≥ 20mm

**UART信号**:
- 走线宽度 ≥ 0.2mm
- 远离高速信号
- 接收端加ESD保护

**关键信号保护**:
- 北斗PPS信号：屏蔽走线
- 风扇PWM：远离敏感电路
- 所有对外接口：TVS管保护

### 6.3 EMC设计

**频段隔离**:
- 5G天线（3.5GHz）与Mesh天线（5.8GHz）间距 ≥ 50mm
- 北斗天线（B1/B2）与通信天线间距 ≥ 30mm

**屏蔽措施**:
- 5G模组：金属屏蔽罩
- Mesh模组：金属屏蔽罩
- 关键电路：π型滤波网络

**接地设计**:
- 完整地平面（第2层）
- 接地孔间距 ≤ λ/20
- 接地电阻 ≤ 0.1Ω
