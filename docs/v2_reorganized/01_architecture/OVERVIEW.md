# 空中空间智能体 - 系统总览

> **核心理念**: "一切皆为智能体" (Everything is an Agent)

---

## 1. 系统定位

**SkyEdge AI 不是"另一个大疆妙算3"，也不是"另一个巨视安全"**

我们构建的是一个 **AI智能体操作系统**，专门为无人机等空中设备提供：

| 能力 | 描述 |
|------|------|
| **空间智能** | 理解三维空间环境，不只是二维图像 |
| **自主决策** | 基于空间记忆和世界模型进行决策 |
| **持续进化** | 从空间交互中持续学习和进化 |
| **开放生态** | 基于多智能体架构的开放生态 |

## 4. 核心技术

### 4.1 五大技术护城河

| 护城河 | 描述 | 状态 |
|--------|------|------|
| **空间记忆与世界模型** | 三维空间地图、语义地图、拓扑地图 | 设计中 |
| **多智能体协同算法** | 拍卖算法、共识算法、冲突解决 | 设计中 |
| **RCMT时空分析** | 时序变化检测，mIoU 78.1% | ✅ 已实现 |
| **智能体操作系统** | 统一接口、生命周期管理 | 开发中 |
| **边缘大模型** | 轻量级VLM，边缘推理 | 规划中 |

### 4.2 RCMT变化检测 (核心算法)

**RCMT (Recurrent Cross-Memory Transformer)** 是我们的核心算法：

- **性能**: mIoU 78.1% vs SOTA 75.2% (+2.9%)
- **能力**: 时序+空间交叉分析，检测环境变化
- **应用**: 城市管理、农业监测、边境巡逻、电力巡检

```

- 进度: Epoch 130/300
- 最佳F1: 0.8924
- 目标: F1 > 0.92
```

## 5. 开发团队协作

### 5.1 智能体分工

| 角色 | 负责模块 | 代码路径 |
|------|----------|----------|
| **软件开发** | 边缘框架、云边平台 | `edge_infer/src/*`, `edge_infer_cloud/backend/*` |
| **算法开发** | RCMT、模型训练 | `edge_infer/models/rcmt/*`, `edge_infer_cloud/models/*` |
| **前端开发** | UI/UX | `edge_infer_cloud/frontend/*` |
| **产品经理** | 需求、路线图 | 文档、规划 |

### 5.2 开发流程

```
1. 需求分析 → 产品经理
2. 技术方案 → 软件开发 + 算法开发
3. 开发实现 → 各自负责模块
4. 集成测试 → 软件开发
5. 模型验证 → 算法开发
6. 部署上线 → 云边协同
```
