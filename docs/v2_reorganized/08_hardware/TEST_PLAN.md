### 3.4 AI推理性能测试

#### 测试模型

- YOLOv8-s (目标检测)
- RCMT V3-Swin (变化检测)

#### 测试方法

```python
# 推理延迟测试
import time
import cv2
from ultralytics import YOLO

model = YOLO('yolov8s.pt')
image = cv2.imread('test.jpg')

# 预热
for _ in range(10):
    model(image)

# 测试100次
times = []
for _ in range(100):
    start = time.time()
    results = model(image)
    end = time.time()
    times.append((end - start) * 1000)  # ms

avg_latency = sum(times) / len(times)
max_latency = max(times)
min_latency = min(times)

print(f"平均延迟: {avg_latency:.2f}ms")
print(f"最大延迟: {max_latency:.2f}ms")
print(f"最小延迟: {min_latency:.2f}ms")
```

#### 性能指标

| 模型 | 输入尺寸 | 平均延迟 | 最大延迟 | FPS | 状态 |
|------|---------|---------|---------|-----|------|
| YOLOv8-s | 640x640 | ≤30ms | ≤50ms | ≥30 | ⬜ |
| RCMT V3 | 512x512 | ≤40ms | ≤60ms | ≥20 | ⬜ |
