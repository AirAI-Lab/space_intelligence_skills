## 三、关键信号走线

### 3.1 USB 3.0差分对（5G模组）

```
设计规则：
- 阻抗：90Ω ±10%（差分）
- 线宽：0.15mm
- 线距：0.1mm
- 等长误差：<5mil
- 走线层：顶层/底层（避开散热片区域）
- 参考层：第2层（GND）

布局示例：

顶层：
  USB-C ───[差分对]─── 5G模组
  (RX+/RX-)  │
             ├─> 等长匹配
             └─> 远离时钟源

ESD保护：
  USB-C ──> TVS管阵列 ──> 差分对
```

### 3.2 UART信号（Mesh/北斗）

```
设计规则：
- 线宽：0.2mm
- 线距：≥0.3mm（3W原则）
- 最大长度：<150mm
- ESD保护：每个信号线加TVS管

信号分配：
  UART1 (Mesh)：
  - TX: GPIO08 ──> Mesh RX (10k上拉)
  - RX: GPIO10 <── Mesh TX (10k上拉)
  
  UART2 (北斗)：
  - TX: GPIO28 ──> 北斗 RX
  - RX: GPIO10 <── 北斗 TX
  - PPS: GPIO35 <── 北斗 PPS (50Ω串联)
```

### 3.3 时钟和复位信号

```
设计规则：
- 时钟线：屏蔽走线，两侧包地
- 复位线：0.2mm线宽，10k上拉
- 远离高速信号 > 3mm

关键时钟：
  - 5G模组：26MHz TCXO
  - 北斗模组：内部TCXO
  - Mesh模组：内部晶振

复位信号：
  - 全局复位：GPIO12 ──> 所有模块RST
  - 5G复位：独立GPIO
  - Mesh复位：独立GPIO
  - 北斗复位：独立GPIO
```

## 五、设计规则

### 5.1 一般设计规则

| 规则项 | 参数 | 说明 |
|--------|------|------|
| **线宽** | | |
| 电源线 | ≥2mm | 5V主电源 |
| 信号线 | ≥0.2mm | 一般信号 |
| 差分对 | 0.15mm | USB 3.0 |
| **间距** | | |
| 信号-信号 | ≥0.15mm | 一般信号 |
| 电源-电源 | ≥0.3mm | 不同电源域 |
| 信号-电源 | ≥0.3mm | 隔离 |
| **过孔** | | |
| 外径 | 0.6mm | 标准过孔 |
| 内径 | 0.3mm | 标准过孔 |
| 间距 | ≥1mm | 过孔阵列 |
| **铜箔** | | |
| 最小铜箔 | 0.2mm | 孤岛铜箔 |
| 网格铜 | 10x10mm | 大面积铺铜 |

### 5.2 高速信号规则

**USB 3.0差分对**：
- 等长匹配：<5mil
- 阻抗控制：90Ω ±10%
- 走线层：顶层优先
- 参考平面：连续GND
- 避免跨越分割区
- 差分对内间距保持一致

**时钟信号**：
- 屏蔽走线（包地）
- 远离高速信号 > 3mm
- 串联电阻靠近源端
- 负载电容 < 10pF

### 5.3 电源设计规则

**主电源走线**：
- 5V电源：线宽 ≥ 2mm
- 地线：线宽 ≥ 2.5mm
- 单点共地（星形接地）
- 接地电阻 ≤ 0.1Ω

**去耦电容**：
- 靠近IC电源引脚
- 多种容值并联
- 最小化回路面积

## 七、EMC设计要点

### 7.1 屏蔽设计

```
屏蔽罩布局：

  ┌─────────────┐
  │ 5G模组     │ <--- 金属屏蔽罩
  │ (MiniPCIe) │      35x50x5mm
  └─────────────┘

  ┌─────────────┐
  │ Mesh模组   │ <--- 金属屏蔽罩
  │ (USB)      │      25x30x5mm
  └─────────────┘

屏蔽罩要求：
- 材质：镀锡铜
- 接地：四角接地 + 边缘多点接地
- 接地电阻：<0.1Ω
```

### 7.2 滤波设计

**电源滤波**：
- 输入端：共模电感 + π型滤波
- 每个电源域：独立π型滤波
- 关键IC：去耦电容阵列

**信号滤波**：
- USB信号：共模电感（选配）
- 时钟信号：磁珠 + 电容
- 复位信号：RC滤波

### 7.3 接地设计

```
接地策略：

1. 星形接地
   - 主电源地 <──┐
   - 5G电源地 <──┤
   - 信号地   <──┤
   - 屏蔽地   <──┘
                 │
            单点共地
             (测试点)

2. 地平面设计
   - 第2层：完整GND平面
   - 接地孔间距：<λ/20
   - 接地孔阵列：散热片下方密集布置

3. 地回路最小化
   - 高速信号紧贴地平面
   - 避免跨越地平面分割
   - 电源回路面积最小化
```
