| 角色 | Emoji | 调用方式 | 核心职责 |

## 🔄 跨角色协作

### 场景1：产品研发

```
负责人 → 制定计划
算法工程师 → 训练模型
硬件工程师 → 硬件支持
软件工程师 → 工程实现
产品工程师 → 编写文档
负责人 → 验收发布
```

### 场景2：问题解决

```
硬件工程师：发现功耗超标
  ↓
软件工程师：优化代码性能
  ↓
算法工程师：压缩模型大小
  ↓
负责人：评估改进效果
```

## 📊 角色职责边界

```
算法工程师: 做什么（What）
  - 模型设计和算法研发

软件工程师: 怎么做（How）
  - 系统架构和工程实现

硬件工程师: 物理层（Physical）
  - 硬件设计和集成

产品工程师: 产品和用户（Product）
  - 功能规划和文档

市场运营: 推广和客户（Marketing）
  - 市场推广和品牌

负责人: 统筹和决策（Leadership）
  - 项目管理和协调
```
