#### 协同算法

**任务分配 (拍卖算法)**:
- 智能体竞价获取任务
- 基于距离、能源、能力的估值
- 支持任务抢占和再分配

**状态同步 (共识算法)**:
- 分布式共识一致性
- Gossip协议信息传播
- 容错节点自动剔除
