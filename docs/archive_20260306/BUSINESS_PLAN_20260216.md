## 3. 产品与服务

#### 3.1.1 硬件产品

**SkyEdge AI Edge Node** - 5G边缘智能一体机

- IMU: 9轴姿态感知
- 散热: 主动散热风扇
- 防护: IP65防水防尘

**性能指标**:
- 推理性能: 100TOPS, 8GB显存
- 5G速率: >100Mbps (下行), >50Mbps (上行)
- RTK精度: 水平<2cm, 垂直<5cm
- Mesh节点: >10
- 温度: <70℃ (满载)
- 重量: <500g (不含天线)
- 尺寸: 150mm×100mm×60mm (不含天线)

**edge_infer** - 边缘推理框架 (开源)

**核心功能**:

**技术栈**:
- 推理引擎: TensorRT 8.5 + FP16量化
- AI模型: YOLOv8 + RCMT
- 通信: MQTT + REST API
- 硬件适配: Jetson Orin Nano

**edge_infer_cloud** - 云边协同平台 (开源)

**核心功能**:

**技术栈**:
- 前端: Vue3 + TypeScript + Element Plus
- 后端: Spring Boot 3.x
- 数据库: PostgreSQL 16 + TimescaleDB
- 缓存: Redis 7.x
- 存储: SeaweedFS

### 3.2 核心功能

#### 3.2.1 边缘AI推理

**功能特性**:
- ✅ 实时视频流推理 (1080P@40FPS)
- ✅ 多模型支持 (YOLOv8/RCMT/检测/分割/姿态)

- 姿态估计: HRNet/MediaPipe
- 语义分割: UPerNet/SegFormer
- OCR识别: PaddleOCR/Tesseract
- 异常检测: Isolation Forest/Autoencoder

#### 3.2.2 云边协同训练

**完整流程**:
1. **数据采集** (边缘端) → 视频流自动采集
2. **数据上传** (云端) → 5G/Mesh自动同步
3. **数据标注** (云端) → AI辅助标注工具
4. **模型训练** (云端) → YOLOv8/RCMT训练
5. **模型评估** (云端) → mAP/F1计算
6. **模型转换** (云端) → .pt→.onnx→.engine
7. **模型部署** (边缘端) → OTA自动推送
8. **模型更新** (边缘端) → 热加载，零停机

**训练能力**:
- ✅ 完整训练 (从头开始)
- ✅ 微调 (基于预训练模型)
- ✅ 续训 (从中断处继续)
- ✅ 迁移学习 (小样本训练)
- ✅ 自动超参优化
- ✅ 分布式训练 (多GPU)

#### 3.2.3 5G/Mesh双模通讯

**通讯模式**:
- **5G模式**: 高带宽、低延迟 (>100Mbps, <20ms)
- **Mesh模式**: 自组织、无中心节点
- **混合模式**: 智能切换，最佳路由
- **离线模式**: 断网自愈，本地推理

**切换策略**:
```python
if 5g_signal > -80dBm:
    use_5g()
elif mesh_neighbors > 0:
    use_mesh()
else:
    use_offline()  # 仅本地推理
```

**性能指标**:
- 5G下载速度: >100Mbps
- 5G上传速度: >50Mbps
- Mesh节点数: >10
- Mesh跳数: <5
- 切换延迟: <1s
