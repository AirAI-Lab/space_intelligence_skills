## 📊 什么是depths参数？

`depths`参数定义了Swin Transformer每个Stage的Transformer Block数量：

```python
# 当前配置 (Swin-Tiny)
depths = [2, 2, 2, 2]
# Stage 1: 2个Transformer blocks
# Stage 2: 2个Transformer blocks
# Stage 3: 2个Transformer blocks
# Stage 4: 2个Transformer blocks
```

## 📈 修改depths对性能的影响

### 1. 增加depth（加深网络）

#### 方案A: 只加深Stage 3 → [2,2,6,2]

**影响分析**:
- **参数量**: 58.72M → ~85M (+45%)
- **计算量**: +60-80%
- **显存**: 3.7GB → ~6-7GB
- **训练速度**: 1.5 it/s → ~0.8-1.0 it/s

**性能预期**:
```
优点：
✅ 特征提取能力更强（Stage 3处理最丰富的语义信息）
✅ 对复杂变化模式的建模能力提升
✅ 变化检测精度可能提升1-3% F1

缺点：
❌ 过拟合风险增加（需要更强的正则化）
❌ 训练时间延长1.5-2倍
❌ 可能需要降低batch_size到8
```

**适用场景**:
- 数据集规模大（>10000样本）
- 变化模式复杂（LEVIR-CD属于中等复杂度）
- 追求极致性能（F1>0.93）

---

- **参数量**: 58.72M → ~95M (+62%)

**性能预期**:
```
优点：
```
