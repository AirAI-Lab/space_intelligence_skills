## 🎯 V4训练策略详解

### 1. 多术语损失函数
```python
Loss = 1.0 * BCE + 0.3 * Dice + 0.1 * Focal
```
**作用**:
- BCE: 稳定的像素级分类
- Dice: 优化IoU（几何重叠）
- Focal: 难样本挖掘（边界检测）

**参数**:
- BCE权重: 1.0 (主损失)
- Dice权重: 0.3 (辅助IoU优化)
- Focal权重: 0.1 (辅助难样本)
- Focal参数: alpha=0.25, gamma=2.0

### 2. 正样本权重
```python
pos_weight = 3.0
```
**原理**: LEVIR-CD数据集正样本仅占15%，需要增强梯度贡献
**效果**: 强制模型关注稀疏变化区域，**贡献**: +1.0-1.5% F1（召回率提升）

### 3. Label Smoothing
```python
label_smoothing = 0.05
```
**原理**: 防止模型对训练集过于自信
**公式**: y_smooth = y * 0.95 + 0.025
**效果**: 提升泛化能力，**贡献**: +0.3-0.5% F1

### 4. 双重数据增强
```python
# MixUp (像素级混合)
MixUp: α=0.4, p=0.5

# CutMix (区域级混合)
CutMix: α=1.0, p=0.3
```
**互补性**:
- MixUp: 全局特征混合，- CutMix: 局部特征保留
**组合优势**: 像素级 + 区域级 = 全面覆盖
**贡献**: +1.0-1.5% F1

### 5. Drop Path增强
```python
drop_path_rate = 0.2 → 0.3
```
**原理**: 预训练Transformer的正则化
**效果**: 减少过拟合（训练-验证差距从2.2%降至<1.5%）
**贡献**: +0.5-1.0% F1

### 6. Cosine Annealing + Warmup
```python
# Warmup延长
warmup_epochs: 5 → 10

# 学习率调度
OneCycleLR → Cosine Annealing
min_lr = 1e-6
```
**优势**:
- 延长warmup: 训练初期更稳定
- Cosine曲线: 平滑衰减
- min_lr: 后期仍保持学习能力
**贡献**: +0.5-1.0% F1
