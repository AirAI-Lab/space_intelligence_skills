Abstract
We present Open-CD, a change detection toolbox that
contains a rich set of change detection methods as well as
related components and modules. The toolbox started from
a series of open source general vision task tools, includ-
ing OpenMMLab Toolkits1, PyTorch Image Models2, etc. It
gradually evolves into a unified platform that covers many
popular change detection methods and contemporary mod-
