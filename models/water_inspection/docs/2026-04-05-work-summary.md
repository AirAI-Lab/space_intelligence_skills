## 🔧 技术细节

### 颜色校验算法
```python
color_dist = np.sqrt(
    (region_bgr[0] - color_hint[0])**2 +
    (region_bgr[1] - color_hint[1])**2 +
    (region_bgr[2] - color_hint[2])**2
) / 441.67

if color_dist > threshold:
    penalty = min(0.5, (color_dist - threshold) * 2)
    adjusted_prob = original_prob * (1.0 - penalty)
```

### 诊断流程
1. 文本对齐检查 (SigLIP2)
2. Patch分割检查 (IoU > 0.5)
3. 颜色一致性检查 (color_dist < threshold)
4. 综合判断: 三者都通过
