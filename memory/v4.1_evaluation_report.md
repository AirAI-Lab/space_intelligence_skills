**当前代码中的类别** (/app/water_inspection/models/open_vocab/radio_segmentor.py:67):
```python
CLASS_COLORS = {
    "black_water": (0, 0, 180),       # 深蓝色
    "brown_water": (42, 42, 165),     # 棕色
    "yellow_water": (0, 255, 255),    # 黄色
    "green_water": (0, 200, 0),       # 绿色
    "red_water": (0, 0, 255),         # 红色
    "milky_water": (200, 200, 200),   # 浅灰色
    "foam_water": (180, 180, 255),    # 淡紫色
    "dam_seepage": (100, 100, 100),   # 深灰色
    "normal_water": (200, 200, 100),  # 淡黄色
}
```

**新的类别定义** (用户提供):
```python
CLASS_COLORS = {
    "black_water": (0, 0, 180),           # 深蓝色 - 黑水

    "red_water": (0, 0, 255),             # 红色 - 红水

}
```

**主要变更**:
1. ❌ 移除 `brown_water`，替换为 `turbid_water` (浑浊水)
