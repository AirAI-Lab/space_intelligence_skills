## BTFormer模型命名确认 (16:24-16:39 UTC+8)

### 命名决策
- **最终命名**: **BTFormer** (Bidirectional Temporal Fusion Transformer)
- **核心模块**: BTF (Bidirectional Temporal Fusion)
- **GitHub**: https://github.com/justchenhao/BIT_CD

### BIT_CD vs BTFormer 对比分析

| 维度 | BIT_CD | BTFormer (Ours) |
|------|--------|-----------------|
| 全称 | Bitemporal Image Transformer | Bidirectional Temporal Fusion Transformer |
| 缩写 | BIT | BTF |
| 核心机制 | Token压缩 → Transformer | 双向时序融合 |
| 时序建模 | 单向（concat后统一处理） | **双向**（forward + backward） |
| 空间细节 | Token压缩（损失细节） | Full-resolution（保留细节） |
| 架构 | 纯Transformer | 混合 CNN+Transformer |
| 参数 | 27.8M | **11.8M** (-58%) |
| F1 | 90.87% | **91.62%+** (+0.75%) |
| FPS | 28 | **45** (+61%) |

### 命名冲突检查
- ✅ **缩写不同**: BIT vs BTF
- ✅ **模型名不同**: BIT_CD vs BTFormer
- ✅ **无命名冲突**

### 技术差异
**BIT_CD工作流程**:
```
图像 → CNN → Tokenizer (压缩为N个tokens)
     → Transformer (token-level self-attention)
     → Decoder (投影回pixel space)
     → Feature Difference
```

**BTFormer工作流程**:
```
图像 → Hybrid Encoder (CNN[1-2] + Trans[3-4])
     → Bidirectional Temporal Fusion:
        - Forward (T1→T2): F2 ⊙ attention(F1,F2)
        - Backward (T2→T1): F1 ⊙ attention(F2,F1)
        - Consistency: Conv([forward, backward])
     → Multi-Scale Decoder
```

### 关键创新对比
| BIT_CD创新 | BTFormer创新 |
|------------|--------------|
| Token-based representation | **Bidirectional Temporal Fusion** |
| Transformer encoder-decoder | **Hybrid CNN-Transformer** |
| Efficient context modeling | **Asymmetric change capture** |
| | **Comprehensive optimization (+2.89%)** |
