# OpenClaw + 智谱 AI 配置指南

## 问题说明

OpenClaw 需要正确配置智谱 AI 的 API 密钥才能使用 glm-4.7 模型。

## 解决方案

### 方式一：使用环境变量（推荐）

创建一个批处理文件来启动 Gateway：

```batch
@echo off
setlocal
REM 设置智谱 AI 环境变量
set ANTHROPIC_AUTH_TOKEN=eccdf9a26cd043a8b9ff470584abec01.Y3DHOQz2wj4RgwbP
set ANTHROPIC_BASE_URL=https://open.bigmodel.cn/api/anthropic

REM 启动 Gateway
cd /d d:\github\edge_infer_cloud
openclaw gateway --port 18789 --token edge-infer-claw-2026
pause
```

### 方式二：使用系统环境变量

在 Windows 中设置用户环境变量：

1. 右键点击"此电脑" → 属性 → 高级系统设置
2. 点击"环境变量"
3. 在"用户变量"中添加：
   - 变量名: `ANTHROPIC_AUTH_TOKEN`
   - 变量值: `eccdf9a26cd043a8b9ff470584abec01.Y3DHOQz2wj4RgwbP`
   - 变量名: `ANTHROPIC_BASE_URL`
   - 变量值: `https://open.bigmodel.cn/api/anthropic`

### 方式三：PowerShell 启动

```powershell
$env:ANTHROPIC_AUTH_TOKEN = "eccdf9a26cd043a8b9ff470584abec01.Y3DHOQz2wj4RgwbP"
$env:ANTHROPIC_BASE_URL = "https://open.bigmodel.cn/api/anthropic"
cd d:\github\edge_infer_cloud
openclaw gateway --port 18789 --token edge-infer-claw-2026
```

## 验证配置

启动后，运行以下命令验证：

```bash
openclaw models list
```

应该看到类似输出：

```
Model                                      Input      Ctx      Local Auth  Tags
anthropic/glm-4.7                          -          -        ✓     -     default,configured
```

注意：Auth 栏应该显示 ✓ 而不是 - 或 missing。

## WhatsApp 登录

确保环境变量设置后，再运行 WhatsApp 登录：

```bash
# 设置环境变量后
openclaw channels login --channel whatsapp
```

## 快速启动脚本

使用已创建的脚本：

1. **启动 Gateway**: 双击 `start-gateway.bat`
2. **登录 WhatsApp**: 双击 `login-whatsapp.bat`

两个脚本都已包含必要的环境变量设置。
