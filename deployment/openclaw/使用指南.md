# OpenClaw + edge_infer_cloud 使用指南

## ✅ 安装完成

### 已配置功能
- **OpenClaw 版本**: 2026.2.3-1
- **AI 模型**: glm-4.7 (智谱 AI)
- **Gateway 地址**: http://localhost:18789
- **工作目录**: d:/github/edge_infer_cloud
- **双重模式**: 聊天机器人 + 自动编程

## 🚀 快速启动

### 方式一：双击脚本（最简单）
1. 进入 `deployment/openclaw/` 目录
2. 双击 `start-gateway.bat`
3. 浏览器自动打开 http://localhost:18789

### 方式二：命令行
```bash
cd d:/github/edge_infer_cloud
openclaw gateway --port 18789 --token edge-infer-claw-2026
```

## 📱 WhatsApp 设置

### 扫码登录

**方式一：双击脚本（推荐）**
```
双击 deployment/openclaw/login-whatsapp.bat
```

**方式二：命令行**
```bash
openclaw channels login
```

1. 终端会显示 QR 码
2. 手机 WhatsApp 扫码
3. 连接成功后即可使用

### 电脑端 WhatsApp
访问 https://web.whatsapp.com 用手机扫码登录

## 🎯 下一步

1. **启动 Gateway**: 双击 `start-gateway.bat`
2. **配置 WhatsApp**: 双击 `login-whatsapp.bat` 或运行 `openclaw channels login`
3. **测试聊天**: 通过 WhatsApp 或 WebChat 发送 "你好"
4. **开始编程**: 尝试 "重启后端服务"

## ⚠️ 注意事项

1. 确保 Gateway 运行后再使用 WhatsApp
2. 手机和电脑需要在同一网络（如需网络功能）
3. 首次使用需要扫码登录 WhatsApp
4. 敏感操作请确认后再执行

## 🆘 遇到问题？

### Gateway 无法启动
```bash
# 检查配置
openclaw doctor

# 停止旧进程
双击 stop-gateway.bat

# 重新启动
双击 start-gateway.bat
```

### WhatsApp 连接失败
```bash
# 重新登录
openclaw channels logout
openclaw channels login
```

### 消息无响应
1. 检查 Gateway 是否运行
2. 发送 `/status` 查看状态
3. 发送 `/new` 重置会话
